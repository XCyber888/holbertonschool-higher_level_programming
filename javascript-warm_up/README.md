# JavaScript - Warm up

## Description
This project is part of the Holberton School curriculum. It covers the basics of JavaScript, including variables, loops, functions, and objects. The goal is to get familiar with the syntax and common practices in Node.js environment while following the `semistandard` coding style.

## Technologies
* Language: JavaScript (Node.js 20.x)
* Style Guide: [semistandard](https://github.com/standard/semistandard)

## Files
| File | Description |
| --- | --- |
| `0-javascript_is_amazing.js` | Prints "JavaScript is amazing" using a constant. |
| `1-multi_languages.js` | Prints 3 lines about different languages. |
| `2-arguments.js` | Prints a message based on the number of arguments passed. |
| `3-value_argument.js` | Prints the first argument passed to the script. |
| `4-concat.js` | Concatenates two arguments in a specific format. |
| `5-to_integer.js` | Converts an argument to an integer. |
| `6-multi_languages_loop.js` | Prints 3 lines using an array and a loop. |
| `7-multi_c.js` | Prints "C is fun" x times based on the first argument. |
| `8-square.js` | Prints a square of 'X' characters. |
| `9-add.js` | Function that adds two integers. |
| `10-factorial.js` | Recursively computes and prints a factorial. |
| `11-second_biggest.js` | Finds the second biggest integer in the list of arguments. |
| `12-object.js` | Updates a value within an object. |
| `13-add.js` | Exported function that returns the addition of 2 integers. |
